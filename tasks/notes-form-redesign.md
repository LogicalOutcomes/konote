# UX-NOTES1: Redesign Detailed Notes Form

**Status:** Design approved
**ID:** UX-NOTES1
**Owner:** GG
**Design doc:** `docs/plans/2026-02-19-notes-form-redesign.md`

## Summary

Redesign the progress notes form into a 4-zone layout with compact target cards, scale pills for 1-5 metrics, auto-calculated metrics, and collapsed optional sections. Prioritises speed for routine entries while keeping full detail available.

## Implementation Phases

- **Phase A:** Layout + field widths (CSS + template only)
- **Phase B:** Scale pills (form + template + CSS)
- **Phase C:** Compact target cards with "No change" shortcut (template + CSS + JS)
- **Phase D:** Auto-calc metrics (model + migration + view + template)

See design doc for full details.
